DSA Core Concepts

Graph Node Conversion: The walkable grid cells are parsed and modeled as vertices in a graph structure. Wall blocks are omitted.

BFS pathfinding: Blinky, Pinky, Inky, and Clyde utilize Breadth-First Search (BFS) to find shortest path sequences. Visited lists and parent backtracking maps operate in real-time.

Ghost Personalities

  • Blinky (Red): Pathfinds directly to Pac-Man's coordinates (direct chaser).
  • Pinky (Pink): Pathfinds to 4 cells ahead of Pac-Man (ambush interceptor).
  • Inky (Cyan): Calculates flanking vector based on Blinky and Pac-Man's vector projection.
  • Clyde (Orange): Runs BFS to Pac-Man when > 8 cells away; otherwise flees back to Clyde's bottom-left scatter target.

Visual Render & Audio

Engine Architecture: Model-View-Controller design pattern separates mathematical physics loop from drawing and HTML5 canvas layers.

Web Audio: Sound effects are synthesized on-the-fly inside the client browser using raw oscillator waves.
